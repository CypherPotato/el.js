# el.js

Essa ferramenta de 2,6 Kb (1,4 Kb gzipped) é uma biblioteca para criar elementos HTML com o máximo de facilidade possível. É uma alternativa funcional para o `document.createElement()`.

## Instalação

Inclua o import pela CDN no `<head>` do seu HTML:

```html
<script src="https://unpkg.com/@cypherpotato/el@0.2.0/dist/latest/el.min.js">
```
